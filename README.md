# TFI 2 Go for Pebble
A pebble watch app to get departure data from Transport For Ireland.
- Bus
  - Dublin Bus
  - Go Ahead
  - Bus Éireann
  - Ulsterbus
- Train
  - Iarnród Éireann
- Tram
  - Luas
